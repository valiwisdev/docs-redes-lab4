## Redes involucradas

### Router R1

- `G0/0`: `2001:ABCD:1435:A1::3/64`
- `G0/1`: `2002:ABCD:1435:B1::3/64`

### Router R2

- `G0/0`: `2001:ABCD:1435:A1::4/64`
- `G0/1`: `2002:ABCD:1435:C1::1/64`

### Router R3

- `G0/0`: `2002:ABCD:1435:B1::4/64`
- `G0/1`: `2002:ABCD:1435:D1::1/64`

## Comandos

### Router R1

```bash
enable
configure terminal
hostname R1
ipv6 unicast-routing
ipv6 router ospf 1
router-id 1.1.1.1
exit
interface <INT_R1_R2>
ipv6 address 2001:ABCD:1435:A1::3/64
ipv6 ospf 1 area 0
no shutdown
exit
interface <INT_R1_R3>
ipv6 address 2002:ABCD:1435:B1::3/64
ipv6 ospf 1 area 0
no shutdown
end
copy running-config startup-config
```

### Router R2

```bash
enable
configure terminal
hostname R2
ipv6 unicast-routing
ipv6 router ospf 1
router-id 2.2.2.2
exit
interface <INT_R2_R1>
ipv6 address 2001:ABCD:1435:A1::4/64
ipv6 ospf 1 area 0
no shutdown
exit
interface <INT_R2_PC1>
ipv6 address 2002:ABCD:1435:C1::1/64
ipv6 ospf 1 area 0
no shutdown
end
copy running-config startup-config
```

### Router R3

```bash

ipv6 router ospf 1
router-id 3.3.3.3
```
