## Comandos para identificar las interfaces de red

```bash
show ipv6 interface brief
show ipv6 interface
show interfaces description
show running-config | section interface
```

## Comandos útiles

Estos atajos son útiles para verificar conectividad `IPv6`, limpiar configuraciones y reiniciar servicios de enrutamiento.

| Acción | Comando de ejemplo |
| --- | --- |
| Guardar configuración | `copy running-config startup-config` |
| Ver rutas IPv6 | `show ipv6 route` |
| Ver protocolos IPv6 activos | `show ipv6 protocols` |
| Reiniciar una interfaz | `interface g0/0`<br/>`shutdown`<br/>`no shutdown` |
| Restaurar una interfaz a valores por defecto | `default interface g0/0` |
| Quitar RIPng del router | `no ipv6 router rip REDLAB` |
| Quitar OSPFv3 proceso 1 | `no ipv6 router ospf 1` |
| Reiniciar el router | `reload` |
| Borrar la configuración guardada y reiniciar | `erase startup-config`<br/>`reload` |

> Reemplace `REDLAB` por el nombre del proceso RIPng que esté usando en la práctica y ajuste `g0/0` según la interfaz correspondiente.
