## Objetivo

Documentar la configuración de `BGP` entre los sistemas autónomos de la topología 2.

## Interfaces dinámicas

Seleccione una sola base de interfaz y se ajustan automáticamente los comandos de R1, R2 y R3.

## Sistemas autónomos

- `R1`: `ASN 100`
- `R2`: `ASN 200`
- `R3`: `ASN 300`

## Redes a anunciar

### Router R1

- `G0/0`: `192.168.10.1/24`
- `G0/1`: `192.168.30.3/24`

### Router R2

- `G0/0`: `192.168.30.23/24`
- `G0/1`: `192.168.40.23/24`
- `G0/2`: `192.168.60.1/24`

### Router R3

- `G0/0`: `192.168.40.3/24`
- `G0/1`: `192.168.50.1/24`
