## Comandos del Laboratorio 5

Los siguientes comandos corresponden al grupo 3, seccion 2. Si el equipo usa `GigabitEthernet` en vez de `FastEthernet`, cambie `f0/x` por la interfaz real del dispositivo. Puede verificarlo con `show interfaces` o `show ip interface brief`.

### Configuracion de PCs

#### PC-A

```text
IP Address: 192.168.5.32
Subnet Mask: 255.255.255.0
Default Gateway: 192.168.5.1
```

#### PC-B

```text
IP Address: 192.168.10.32
Subnet Mask: 255.255.255.0
Default Gateway: 192.168.10.1
```

#### PC-C

```text
IP Address: 192.168.15.32
Subnet Mask: 255.255.255.0
Default Gateway: 192.168.15.1
```

### R1

```bash
enable
configure terminal
hostname R1

line console 0
logging synchronous
exit

interface g0/0
no shutdown
exit

interface g0/0.1
encapsulation dot1Q 1
ip address 192.168.1.1 255.255.255.0
exit

interface g0/0.5
encapsulation dot1Q 5
ip address 192.168.5.1 255.255.255.0
exit

interface g0/0.10
encapsulation dot1Q 10
ip address 192.168.10.1 255.255.255.0
exit

interface g0/0.15
encapsulation dot1Q 15
ip address 192.168.15.1 255.255.255.0
exit

end
copy running-config startup-config
```

#### Verificacion en R1

```bash
show ip interface brief
show ip route
show running-config
```

### S1

```bash
enable
configure terminal
hostname S1

line console 0
logging synchronous
exit

interface vlan 1
ip address 192.168.1.5 255.255.255.0
no shutdown
exit

ip default-gateway 192.168.1.1

vlan 5
name Estudiantes32
exit

vlan 10
name Docentes32
exit

vlan 15
name Administrativos32
exit

interface f0/1
switchport mode trunk
no shutdown
exit

interface f0/2
switchport mode trunk
no shutdown
exit

interface f0/5
switchport mode access
switchport access vlan 5
no shutdown
exit

end
copy running-config startup-config
```

#### Verificacion en S1

```bash
show vlan brief
show interfaces trunk
show running-config
```
