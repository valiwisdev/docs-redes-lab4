## Comandos del Laboratorio 5

### Configuración de PCs

#### PC-A

```text
IP Address: 192.168.5.32
Subnet Mask: 255.255.255.0
Default Gateway: 192.168.5.1
```

#### PC-B

```text
IP Address: 192.168.10.32
Subnet Mask: 255.255.255.0
Default Gateway: 192.168.10.1
```

#### PC-C

```text
IP Address: 192.168.15.32
Subnet Mask: 255.255.255.0
Default Gateway: 192.168.15.1
```

#### Verificación en R1

```bash
show ip interface brief
show ip route
show running-config
```

### S1

```bash
enable
configure terminal
hostname S1

line console 0
logging synchronous
exit

interface vlan 1
ip address 192.168.1.5 255.255.255.0
no shutdown
exit

ip default-gateway 192.168.1.1

vlan 5
name Estudiantes32
exit

vlan 10
name Docentes32
exit

vlan 15
name Administrativos32
exit

interface f0/1
switchport mode trunk
no shutdown
exit

interface f0/2
switchport mode trunk
no shutdown
exit

interface f0/5
switchport mode access
switchport access vlan 5
no shutdown
exit

end
copy running-config startup-config
```

#### Verificación en S1

```bash
show vlan brief
show interfaces trunk
show running-config
```

```bash

ping 192.168.1.1
```

Desde PC-B:



### Exportar configuración para entrega

En cada switch y en el router ejecute:

```bash
show running-config
```

Copie el resultado en un archivo de texto y tome captura de pantalla, según lo solicita el laboratorio.
