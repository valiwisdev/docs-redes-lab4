## Objetivo

Documentar la configuración de `OSPF` para los routers involucrados en la topología 1.

## Interfaces dinámicas

Seleccione una sola base de interfaz y se ajustan automáticamente los comandos de R1, R2 y R3.

## Redes a anunciar

### Router R1

- `G0/0`: `192.168.10.3/24`
- `G0/1`: `192.168.20.3/24`

### Router R2

- `G0/0`: `192.168.10.4/24`
- `G0/1`: `192.168.30.1/24`
- `Loopback 2`: `172.16.2.3/16`

### Router R3

- `G0/0`: `192.168.20.4/24`
- `G0/1`: `192.168.40.1/24`
- `Loopback 3`: `172.17.2.3/16`

## Verificación

```txt
PC1> ping 192.168.10.3   - Verifica la conectividad entre PC1 y la interfaz de R1
PC1> ping 192.168.30.1   - Confirma que PC1 alcanza la red conectada a R2
PC1> ping 192.168.40.1   - Comprueba que PC1 puede llegar hasta la interfaz LAN de R3 por medio de OSPF
PC1> ping 192.168.40.32  - Valida el acceso a un host remoto dentro de la red de PC2

PC2> ping 192.168.20.3   - Verifica la conectividad entre PC2 y la interfaz de R1
PC2> ping 192.168.30.1   - Confirma que PC2 alcanza la red conectada a R2
PC2> ping 192.168.40.1   - Comprueba la conectividad local hacia su router de acceso R3
PC2> ping 192.168.30.32  - Valida el acceso a un host remoto dentro de la red de PC1
```
