## Comandos útiles

Use estos comandos cuando necesite ajustar rápidamente la configuración del router durante la práctica.

| Acción | Comando de ejemplo |
| --- | --- |
| Guardar configuración | `copy running-config startup-config` |
| Ver rutas aprendidas | `show ip route` |
| Ver protocolos activos | `show ip protocols` |
| Reiniciar una interfaz | `interface g0/0`<br/>`shutdown`<br/>`no shutdown` |
| Restaurar una interfaz a valores por defecto | `default interface g0/0` |
| Quitar RIPv1 del router | `no router rip` |
| Quitar OSPF proceso 1 | `no router ospf 1` |
| Reiniciar el router | `reload` |
| Borrar la configuración guardada y reiniciar | `erase startup-config`<br/>`reload` |

> Cambie `g0/0` por la interfaz que corresponda. Use `erase startup-config` solo si desea dejar el router limpio para empezar de nuevo.
