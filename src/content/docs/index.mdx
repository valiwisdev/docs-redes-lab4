## Ruta sugerida

1. Revise primero el [material de apoyo](/material-apoyo/).
2. Abra la topologia que corresponda a su practica.
3. Use el resumen para validar direccionamiento e interfaces.
4. Entre al protocolo especifico para copiar o adaptar la configuracion.
5. Descargue el archivo de Packet Tracer y, si existe, las capturas de Wireshark.
